Для iOS используем EAS Build (не локально, т.к. нет Mac). Нужно настроить app.json и запустить.

  • ☐ Получить Project ID на expo.dev/accounts/seydi_123/projects/shapak-translator → Settings → Project ID (UUID)
  • ☐ Прописать в app.json: owner: "Seydi_123" и extra.eas.projectId: "<UUID>"
  • ☐ Поднять ios.buildNumber в app.json (сейчас "1.0.0", надо выше последнего опубликованного в App Store Connect)
  • ☐ Установить EAS CLI: npm install -g eas-cli (если ещё нет)
  • ☐ eas login — войти под Seydi_123
  • ☐ Сборка: eas build --platform ios --profile production (~15-30 мин в облаке)
  • ☐ Скачать .ipa или использовать eas submit --platform ios для автоматической отправки в App Store Connect
  • ☐ В App Store Connect: заполнить release notes, обновить Privacy Policy URL (тот же https://shapak-apps.github.io/turkmen-phrasebook/privacy-policy.html) + Data Deletion URL если требуется
